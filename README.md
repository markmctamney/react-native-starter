# React Native Starter

## Installation

```
npx create-react-native-app --template https://github.com/markmctamney/react-native-starter
yarn && yarn pods
```

## Android

### React Native

```
yarn android
```

### Expo

```
yarn android:expo
```

## iOS

### React Native

```
yarn ios
```

### Expo

```
yarn ios:expo
```

## Web

```
yarn web
```
